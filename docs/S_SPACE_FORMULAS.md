## 一、公理体系

### 公理 1：状态空间

模型在第 $l$ 层的内部表示是 $\mathbb{R}^d$ 中的一个点：

$$s(l) \in \mathbb{R}^d \quad (d = 1024 \text{ for 0.8B model})$$

### 公理 2：仿射结构（特殊 S 理论）

S-Space 是**精确的仿射空间**。对同一层任意三个类型状态：

$$\Delta(A \to C) = \Delta(A \to B) + \Delta(B \to C) \quad \text{[EXACT]}$$

其中 $\Delta(A \to B) = \bar{s}_B(l) - \bar{s}_A(l)$

> **验证**：使用未归一化位移，可加性 cos = 1.0000，曲率 |R| = 0.0000。
> 这是平凡的（向量减法定义所保证），但意义深远——**均态空间是平的**。

**推论**：导航 = 向量算术

$$s_{target} = s_{current} + \Delta(current \to target)$$

### 公理 3：度量张量

两状态间的平方距离：

$$d^2(s_A, s_B; l) = \sum_{k=1}^{d} g_k(l) \cdot [\hat{e}_k(l) \cdot (s_A - s_B)]^2$$

其中：
- $\hat{e}_k(l)$ = 第 $l$ 层位移空间的第 $k$ 主方向
- $g_k(l) = S_k^2(l) / n$ = 度量权重（$S_k$ 为 SVD 奇异值）

**关键数值**：
| 层 | Top-3 度量权重 | Top-10 度量权重 | 有效秩 |
|----|---------------|----------------|--------|
| L3  | 32.5% | 65.2% | 30.8 |
| L19 | 40.3% | 76.4% | 27.5 |

**推论**：导航只需在 Top-K（K≈10）主方向上进行，即可捕获 65-76% 的结构。

### 公理 4：层膨胀定律

层间类型距离呈指数膨胀：

$$\bar{d}(l) \approx A \cdot e^{\alpha \cdot l} \quad \text{for } l \geq 14$$

拟合参数：$A = 0.205$, $\alpha = 0.092$ / 层

**三阶段结构**：
```
L0    : 嵌入阶段   d̄ = 0.98  (token embedding，距离大)
L1-11 : 压缩阶段   d̄ ≈ 0.5-0.6  (表示坍缩，类型难以区分)
L12-14: 过渡阶段   d̄ 开始膨胀
L15-23: 分化阶段   d̄ = 0.8→2.5  (指数膨胀，类型快速分化)
```

**膨胀因子**：

$$\text{scale}(l) = \frac{\bar{d}(l)}{\bar{d}(L3)}$$

| 层 | d̄ | scale |
|----|-----|-------|
| L3  | 0.59 | 1.00 |
| L14 | 0.81 | 1.36 |
| L19 | 1.44 | 2.44 |
| L23 | 2.52 | 4.27 |

### 公理 5：归一化曲率（一般 S 理论）

当位移被归一化为单位向量时：

$$\hat{\Delta}(A \to B) = \frac{\Delta(A \to B)}{|\Delta(A \to B)|}$$

可加性退化：$\cos(\hat{\Delta}_{AB} + \hat{\Delta}_{BC}, \hat{\Delta}_{AC}) \approx 0.95$

**曲率来源**：归一化将平直的仿射空间投影到单位球面上，引入伪曲率：

$$R_{norm}(A,B,C) = \hat{\Delta}_{AB} + \hat{\Delta}_{BC} - \hat{\Delta}_{AC}$$

| 层 | |R_true| | |R_norm| | 放大倍数 |
|----|---------|----------|---------|
| L3  | 0.000 | 0.239 | ∞ |
| L19 | 0.000 | 0.351 | ∞ |

> **之前的所有实验都使用了归一化位移** → 我们一直在曲面上导航，
> 这解释了分类器坍缩和导航不稳定的部分原因。

## 二、核心公式

### 公式 1：导航方程

$$s_{target}(l) = s_{current}(l) + \sum_{k=1}^{K} \alpha_k(l) \cdot \hat{e}_k(l)$$

其中 $\alpha_k(l) = \hat{e}_k(l) \cdot \Delta_{needed}(l)$

- $K = 10$ 足以捕获 65-76% 的导航精度
- $\Delta_{needed}(l)$ 由小脑根据反馈信号计算

### 公式 2：小脑注入方程

$$\text{inject}(l) = P_l \cdot \Delta_{needed}(l) \cdot \text{scale}(l)$$

其中：
- $P_l$ = 投影到 Top-K 主方向的算子
- $\Delta_{needed}(l)$ = 所需位移（来自 need_doing_gap 或闭环反馈）
- $\text{scale}(l) = \bar{d}(l) / \bar{d}(L3)$ = 层依赖膨胀因子

**约束**：
1. **必须使用未归一化位移**，否则引入 ~20% 伪曲率
2. **主方向逐层漂移**（cos² ≈ 0.6 跨层），不能跨层复用 $\hat{e}_k$
3. **膨胀因子必须补偿**，否则深层注入不足

### 公式 3：距离方程

$$d(s_A, s_B; l) = \|s_A(l) - s_B(l)\|_2 = \sqrt{\sum_{k=1}^{d} g_k(l) \cdot \delta_k^2}$$

其中 $\delta_k = \hat{e}_k(l) \cdot (s_A - s_B)$

### 公式 4：层间演化方程

$$s(l+1) = s(l) + r(l)$$

其中 $r(l)$ 是第 $l$ 层的残差贡献。

类型均值状态跨层演化：

$$\bar{s}_T(l+1) = \bar{s}_T(l) + \bar{r}_T(l)$$

**膨胀的来源**：深层 $|r(l)|$ 增大 → 距离膨胀

## 三、关键洞见

### 3.1 为什么分类器坍缩？

L19 处类型间余弦相似度极高：
- 代码↔常识 cos = 0.933
- 对话↔常识 cos = 0.936
- 历史↔对话 cos = 0.943

但真实距离却不同：
- 代码↔常识 = 0.44
- 对话↔常识 = 0.50
- 历史↔对话 = 0.35（近）vs 因果推理↔情感 = 1.43（远）

**解释**：类型位于相似的方向（从原点看），但在不同的径向距离上，且具有微妙的正交分量。分类器（本质上是径向的）无法区分同方向不同距离的向量。

### 3.2 为什么之前的注入方法不稳定？

1. **归一化丢失度量**：所有位移被归一化到 1.0，丢失了真实的距离信息
2. **伪曲率**：归一化引入 20-35% 的伪曲率，导航在曲面上进行
3. **跨层复用方向**：主方向跨层漂移（cos² ≈ 0.6），不能直接跨层复用
4. **忽略膨胀因子**：深层需要 2-4× 的注入强度

### 3.3 为什么 14 类型 PCA 只找到 3 维？

14 个类型方向是从 35 类均值中提取的 14 个特定向量，它们本身高度相关（常识=知识 cos=1.0）。这 14 个向量只覆盖了位移空间的一个**低维子空间**。完整位移空间的有效维度是 28-31，远大于 3。

> **14 类型分类框架只看到了 S-Space 的一个角落。**

## 四、与相对论的类比

| 物理学 | S-Space |
|--------|---------|
| 闵可夫斯基时空（平直） | 未归一化位移空间（精确仿射） |
| 广义相对论（弯曲） | 归一化位移空间（单位球面投影） |
| 度规张量 $g_{\mu\nu}$ | 主方向度量权重 $g_k(l)$ |
| 光速不变原理 | 仿射结构不变（可加性精确成立） |
| 引力 = 时空弯曲 | 归一化 = 仿射→球面投影 |
| 测地线方程 | 导航方程 |
| 等效原理 | 局部（单层）看是平的，全局（跨层）看是弯曲的 |

**S-Space 的"光速不变"**：无论哪三个类型、哪一层，Δ(A→C) = Δ(A→B) + Δ(B→C) 始终精确成立。

**S-Space 的"引力"**：归一化操作，将平直空间投影到球面，制造了伪曲率。

## 五、待验证预言

1. **使用未归一化位移注入应比归一化位移更稳定**（消除 20% 伪曲率）
2. **投影到 Top-10 主方向后注入，应比全维注入更鲁棒**（降噪 + 聚焦结构维度）
3. **注入强度随层膨胀因子 scale(l) 调整，应减少深层注入不足**
4. **跨层不复用方向向量，每层使用独立的主方向，应提高导航精度**
5. **在 Top-3 主方向空间中，类型间可能存在可分的线性边界**（因为 40% 的度量集中在这 3 维）

## 六、局限性

1. **仿射结构仅对类型均值成立**：$\Delta = \bar{s}_B - \bar{s}_A$，对单个样本的位移可能有偏差
2. **模型响应非线性**：即使均态空间平坦，模型对注入的响应仍是非线性的
3. **归一化在实践中可能必要**：如果不同类型位移幅度差异过大，不归一化可能导致数值不稳定
4. **有效维度 28-31 可能偏高**：实际导航可能需要更多方向，Top-10 只覆盖 65-76%
5. **跨层漂移未建模**：主方向如何逐层漂移缺乏数学描述
