## 四大模块

### ① 终端控制系统 (CLI Shell) — 开发者入口

| 命令 | 状态 |
|------|------|
| `lottery extract`              | ✅ 三种提取方式 (LoRA/Phase1/Diff) |
| `lottery compile --intent`     | ❌ **核心新增** |
| `lottery inject`               | ✅ |
| `lottery triggers`             | ✅ |
| `lottery quantify`             | ✅ |
| `lottery gcg`                  | ✅ |
| `lottery blackbox`             | ✅ |
| `lottery demo`                 | ✅ |
| `lottery config`               | ✅ |

### ② 元语言编译器 (Meta-Language Compiler) ★ 核心创新

**输入**：自然语言意图 → **输出**：可直接部署的触发指令

```
用户输入 "悬疑、压抑、步步紧逼"
  Step 1: LLM拆解意图 → 生成对比提示词对
  Step 2: 自动提取彩票方向 (Phase1/LoRA/SVD)
  Step 3: 词表投影 → top-50触发词
  Step 4: 可选GCG搜索 → 最优后缀
  Step 5: 输出指令 + 置信度
```

### ③ 本地彩票提取器

三种方式，自动调度：
- **LoRA扫描** (5秒) — 最快最精确
- **激活对比** (30秒) — 零门槛，无需额外数据
- **模型差SVD** (5分) — 全面覆盖

### ④ 黑盒验证器

- CLI版: `lottery blackbox --suffix "..."` ✅
- Web UI版: 粘贴即测，自动检测模式切换 🔧
