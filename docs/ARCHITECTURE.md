## 核心模块

### ① 三公式引擎 (formulas.py) — 全部核心

| 公式 | 函数 | 模式 |
|------|------|------|
| ③ 定位 | `read_coords(h, ê_k)` | 通用 |
| ② 导航 | `compute_delta(coords, g_k)` | 惯性模式（默认） |
| ② 导航 | `compute_delta(coords, g_k, d_con, d_mag, d_conf)` | 共识模式（可选） |
| ① 力度 | `compute_injection(h, delta_k, ê_k, g_k, r_eff)` | 通用 |

### ② 导航器 (navigator.py) — 生产级

- 自动检测注入层（从PCA数据中选早/中/晚三层）
- 自动计算层间scale（从度量权重分布推导）
- 惯性模式/共识模式自动切换
- 目标注册 + 持续导航 + 漂移修正

### ③ 几何结构 (space.py) — 数学公理

- 仿射空间验证（cos=1.0000）
- 度量张量 g_k = S_k²/n
- 层膨胀律 d̄ ≈ A·e^(0.092l)

### ④ 注入掩码 (injection_mask.py) — 可选安全层

- 从度量权重生成掩码
- 从轴索引生成掩码
- 默认不掩码，need_k自动选轴

## 两种导航模式

| | 惯性模式 | 共识模式 |
|---|---------|---------|
| **d_k来源** | iw_k × c_k（坐标惯性） | d_consensus × d_magnitude × d_confidence |
| **需要什么数据** | 仅ê_k + g_k | ê_k + g_k + 预提取方向数据 |
| **适用范围** | 任何Transformer | 有提取数据的模型 |
| **核心机制** | 跟着坐标走，饱和轴少推 | 沿预设推理方向导航 |
| **安装门槛** | 装上即用 | 需先提取方向数据 |
